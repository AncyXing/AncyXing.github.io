验证

验证class文件是否符合规范

文件格式校验

  • 是否是0xCAFEBABE开头
  • 版本号是否合理

元数据校验

  • 是否有父类
  • 是否继承了final类
  • 非抽象类是否实现了抽象类的所有抽象方法

字节码验证

  • 运行检查
  • 跳转指令是否指向合理的位置

符号引用验证

  • 常量池中描述类是否存在
  • 访问的方法或字段是否存在且有足够的权限

-Xverify:none关闭验证，可以加快启动速度

  • 初始化的顺序和源文件中的顺序一致
  • 如果有父类，会先调用父类的类构造器方法
